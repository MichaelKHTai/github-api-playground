# github-api-playground
 
# Project Title

Github API Playground

### Prerequisite

Install `pipenv`
```
https://github.com/pypa/pipenv
```


### Installing

```
pipenv sync
```

## Getting Started

```
pipenv shell
python main.py
```

## Optional Argument
```
python main.py --detail
```
For saving the full json data in local.
